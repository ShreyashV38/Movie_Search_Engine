# Movie Search Engine
[![Ask DeepWiki](https://devin.ai/assets/askdeepwiki.png)](https://deepwiki.com/ShreyashV38/Movie_Search_Engine)

This is a simple front-end application built with React and Vite that allows users to search for movies from a predefined list. The search results are displayed in real-time as the user types into the search bar.

## Features

- **Movie Search**: A search bar to filter movies by title.
- **Dynamic Filtering**: The list of movies updates instantly as you type.
- **Card-Based UI**: Movies are displayed in a clean, card-based grid layout.
- **Favorite Button**: Each movie card includes a button to mark it as a favorite (currently alerts on click).

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast front-end build tool for modern web projects.
- **ESLint**: For identifying and reporting on patterns found in ECMAScript/JavaScript code.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have Node.js and npm installed on your machine.

- [Node.js](https://nodejs.org/) (which includes npm)

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/shreyashv38/movie_search_engine.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd movie_search_engine
    ```
3.  **Install NPM packages:**
    ```sh
    npm install
    ```
4.  **Start the development server:**
    ```sh
    npm run dev
    ```
    The application will be available at `http://localhost:5173` (or the next available port).

## Available Scripts

In the project directory, you can run:

-   `npm run dev`: Runs the app in development mode.
-   `npm run build`: Builds the app for production to the `dist` folder.
-   `npm run lint`: Lints the source files using ESLint.
-   `npm run preview`: Serves the production build locally to preview it.
